=== Log Analytics output plugin

include::{include_path}/plugin_header.asciidoc[]

==== Description

A plugin for Oracle Cloud Infrastructure (OCI) Log Analytics service.
This plugin groups incoming log events from Logstash by their log group ID,
chunking them into smaller batches of maximum 2 MB size and uploads it as a ZIP payload to OCI Log Analytics.

[id="plugins-{type}s-{plugin}-auth_type"]
===== `auth_type` 

  * Value type is <<string,string>>
  * Default value is `"InstancePrincipal"`

Authentication type for Log Analytics client. Available options are: `"InstancePrincipal"` and `"ConfigFile"`.
If `"InstancePrincipal"` is used, the client will initialize with an Instance Principals Security Token Signer.
If `"ConfigFile"` is used, the client will initialize with the given Config File especified in <<plugins-{type}s-{plugin}-config_file_location>>.
